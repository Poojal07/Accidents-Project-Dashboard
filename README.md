# Accidents-Project-Dashboard
An interactive Power BI dashboard analyzing road accident data across India. It highlights accident causes, severity, vehicle types, weather impact, and time-based trends. The dashboard provides state and city-level insights with KPIs, filters, and visuals to support data-driven road safety analysis and decision-making.
